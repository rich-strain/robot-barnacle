Portfolio Site — Richard Strain
Project Overview
Personal developer portfolio site for Richard Strain, a full-stack web developer and UCF coding bootcamp graduate targeting remote junior to mid-level positions.
Hosting & Deployment

Hosted on Netlify
DNS managed through Cloudflare
Connected to GitHub repo — auto-deploys on push to main branch

Tech Stack

Review existing codebase and note framework before making changes
Resend API for contact form email handling
Netlify serverless functions for backend API calls

Key Goals

Professional, modern, dark mode design
Represent a full-stack developer with payment integration experience
Mobile responsive
Fast and production ready

Important Notes

Do not use stock photos or images of people
Do not expose API keys on the frontend — use Netlify environment variables
Keep commits clean and organized
Resend API key stored as NETLIFY environment variable: RESEND_API_KEY

## Environment

- Node.js version: 22
- .node-version file in root specifies Node version for Netlify
